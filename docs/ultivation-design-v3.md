### 1.4 灵根系统

**测灵根触发时机：** 第一次与七玄门「灵根鉴定石板」交互，永久写入 PlayerData。

灵根分为天灵根、异灵根、双灵根、三灵根、伪灵根、杂灵根。

天灵根：修炼速度 ×5.0，灵力回复 ×2.0，突破成功率 +25%。  
异灵根：修炼速度 ×4.0，灵力回复 ×1.8，突破成功率 +18%。  
双灵根：修炼速度 ×3.0，灵力回复 ×1.5，突破成功率 +10%。  
三灵根：修炼速度 ×2.0，灵力回复 ×1.2，突破成功率无加成。  
伪灵根：修炼速度 ×1.0，灵力回复 ×1.0，突破成功率 -10%，丹药吸收率 ×1.15。  
杂灵根：修炼速度 ×0.8，灵力回复 ×0.8，突破成功率 -18%，丹药吸收率 ×1.25。

低资质灵根修炼慢，但更依赖丹药与机缘。伪灵根和杂灵根更容易获得青玉小瓶。青玉小瓶可以凝聚灵液，灵液能够催熟灵草、提升年份、辅助炼丹，是低资质玩家逆天改命的核心道具。

> **补偿逻辑：** 伪灵根修炼慢，但神秘小瓶产出的丹药/材料价值远超差距，走「炼丹型」差异化路线。

## 二、境界与突破系统

### 2.1 境界突破流程

```
[修为值 ≥ 当前层上限]
  → 玩家手动触发「突破」（按键/GUI）
    → 检查突破资源（丹药/材料）
      → 计算突破成功率
        → 成功：境界+1，修为清零，获得突破属性加成
        → 失败：修为回退 20%，走火风险+10%
          → 走火风险 ≥ 70%：触发走火入魔检定
```

### 2.2 关键突破资源

| 突破节点 | 必需材料 | 稀有度 | 基础成功率 |
|---------|---------|-------|---------|
| 练气各层 | 凝气丹（一阶） | 普通 | 90% |
| **练气13→筑基** | **筑基丹（二阶，极稀缺）** | **极稀有** | 60%（可提升）|
| 筑基各阶 | 筑基期对应草药 | 稀有 | 75% |
| 金丹期 | 金丹草（三阶）+ 特殊辅材 | 稀有 | 55%（金丹颜色由此决定）|
| 元婴期 | 化婴丹（四阶）+ 元婴草 | 极稀有 | 40% |
| 化神期 | 化神丹 + 大道雷光石 | 极稀有 | 35% |
| 渡劫期 | 渡劫丹 + 天雷台建筑 | 任务专属 | 见§渡劫机制 |

**成功率提升方式：**
- 服用辅助丹药：`+5% ~ +20%`
- 在地灵之眼修炼：`+15%`
- 功法品质加成：`+0% ~ +10%`（天阶功法最高）
- 连续失败积累「执念」：每次失败+5%（最多累计30%）

### 2.4 走火入魔机制【MVP】

**触发条件（以下任一）：**
- `走火风险 ≥ 70%` 时突破失败
- 修炼时受到伤害（受伤状态下修炼每秒+2%走火风险）
- 使用超出当前境界 2 级以上的功法

**走火判定表：**

| 走火风险值 | 触发后效果 |
|----------|----------|
| 70~79% | 轻微走火：损失 30% 当前修为，走火风险清零 |
| 80~89% | 中度走火：损失 50% 修为 + 昏迷 30 秒，走火风险清零 |
| 90~99% | 严重走火：掉落一个境界 + 昏迷 3 分钟，装备随机损坏 |
| 100% | **极端走火：当场死亡，背包掉落 50%** |

**降低走火风险方式：**
- 服用「稳神丹」：-20%
- 平稳打坐（不使用技能的打坐状态）：每小时-5%
- 在天然丹炉地打坐：额外每小时-10%

## 四、战斗系统

### 4.1 伤害计算公式

```
最终伤害 = 技能基础伤害
         × (1 + 攻击力加成/100)
         × 属性克制系数（同灵根属性技能：×1.3；克制属性：×1.5；被克制：×0.7）
         × 境界压制系数（攻击方高2+级：×1.3；低2+级：×0.6）
         × 暴击（5%概率×2.0）
         - 目标防御力
```

## 五、生产系统

### 5.1 炼丹系统【MVP】

**炼丹流程：**
```
丹炉（方块）→ 放入丹方（图纸）+ 灵草材料
  → 催火（消耗灵力或灵火石）→ 等待时间
    → 判定（技能等级 + 材料新鲜度 + 灵地加成）
      → 成功：产出丹药（品质随机：下品/中品/上品）
      → 失败：产出废丹 / 爆炉（小概率，丹炉损耗）
```

**原著核心丹药列表：**

| 丹药名 | 品阶 | 主要材料 | 用途 |
|--------|------|---------|------|
| 凝气丹 | 一阶 | 浮云草×3 + 碧云草×1 | 练气期修炼加速（服用后1h修为×2）|
| **筑基丹** | **二阶** | **万年灵草×1 + 血灵芝×2 + 天灵草×1** | **练气→筑基突破（全游最稀缺丹）**|
| 稳神丹 | 一阶 | 定神草×2 | 走火风险-20% |
| 回灵丹 | 二阶 | 回灵草×3 + 玉露×1 | 立刻恢复50%灵力 |

### 9.4 副本/秘境实现方案

```java
// 副本入口：复制预设结构到随机空白坐标（防止污染主世界）
public class DungeonManager {
    private static final Map<UUID, BlockPos> activeInstances = new HashMap<>();

    public static BlockPos enterDungeon(Player player, ResourceLocation templateId) {
        StructureTemplate template = level.getStructureManager().get(templateId);
        BlockPos instancePos = findEmptySpace(level, 5000, 5000); // 在远离玩家的位置
        template.placeInWorld(level, instancePos, new StructurePlaceSettings());
        activeInstances.put(player.getUUID(), instancePos);
        teleportPlayer(player, instancePos.offset(5,1,5)); // 传送到副本入口
        return instancePos;
    }

    public static void exitDungeon(Player player) {
        BlockPos pos = activeInstances.remove(player.getUUID());
        if (pos != null) scheduleCleanup(pos, 600); // 10分钟后删除区块
        teleportPlayer(player, player.getRespawnPosition()); // 返回副本入口
    }
}
```
