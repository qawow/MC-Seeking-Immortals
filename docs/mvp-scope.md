# MVP Scope — Phase 1 最小可行产品范围

> 基于 `ultivation-design-v3.md` 提取的阶段一 MVP 内容
> 
> **原则：** 只做核心可玩循环，不做扩展内容，不实现代码

---

## 一、核心目标

构建最小可玩的修仙模组循环：

```
玩家进入游戏 → 测灵根 → 打坐修炼 → 学习技能 → 战斗/生产 → 突破境界 → 加入宗门 → 获取资源 → 循环强化
```

**验收标准：** 玩家可以从凡人修炼到练气13层，学会基础技能，炼制丹药，加入宗门，体验完整的七玄门任务线。

---

## 二、MVP 包含内容

### 2.1 核心属性系统

✅ **六大核心属性（存储于 PlayerData NBT）**
- 修为 (`cultivation` / `cultivationMax`)
- 灵力 (`mana` / `manaMax`)
- 神识 (`divSense`)
- 肉身强度 (`bodyRef`)
- 走火风险 (`qiDevRisk`)
- 天劫承受 (`tribRes`) — **仅字段预留，MVP不实现天劫**

✅ **战斗衍生属性（运行时计算）**
- 攻击力、防御力、最大HP、灵力回速、修为回速、飞行速度

✅ **境界系统**
- 13层练气（练气1~13）
- 4阶筑基（筑基初/中/后/圆满）
- **MVP 仅实现到筑基圆满**，金丹及以上仅预留数据结构

✅ **灵根系统**
- 六种灵根类型：天灵根、异灵根、双灵根、三灵根、伪灵根、杂灵根
- 永久性测试（墨老先生 NPC 触发）
- 修炼速度、灵力回复、突破成功率、丹药吸收率差异

### 2.2 修炼与突破机制

✅ **打坐修炼系统**
- 打坐 GUI 界面
- 实时修为增长计算：`境界基准 × 灵根系数 × 打坐加成 × 灵脉加成`
- 修炼时受伤打断机制
- 饥饿度消耗

✅ **突破系统**
- 手动触发突破（按键/GUI）
- 突破资源检查（丹药/材料）
- 成功率计算（基础成功率 + 灵根加成 + 丹药加成 + 地点加成）
- 突破成功：境界+1，修为清零，属性加成
- 突破失败：修为回退20%，走火风险+10%

✅ **走火入魔机制【核心 MVP】**
- 四级走火判定（70-79% / 80-89% / 90-99% / 100%）
- 触发条件：突破失败、修炼时受伤、使用超阶功法
- 降低方式：稳神丹、平稳打坐、特殊地点

### 2.3 战斗系统

✅ **练气期技能（9个）**
| 技能名 | 解锁境界 | 类型 |
|--------|---------|------|
| 引气入体 | 练气1 | 被动 |
| 火球术 | 练气3 | 攻击 |
| 冰锥术 | 练气3 | 攻击 |
| 雷击术 | 练气3 | 攻击 |
| 土遁步 | 练气4 | 移动 |
| 御剑飞行(初) | 练气7 | 移动 |
| 单剑刺击 | 练气7 | 攻击 |
| 灵气探测 | 练气10 | 辅助 |
| 三才剑阵 | 练气13 | 攻击 |

✅ **筑基期技能（6个）**
- 神识扩展（被动）
- 御剑飞行（进阶）
- 罡气护体
- 五行遁术
- 北斗剑阵
- 阵法感知（被动）

✅ **飞剑系统**
- 单剑 → 三剑 → 七剑 阵法
- 飞剑实体（Custom Entity for riding, Projectile for attacks）
- 飞剑耐久与存储

✅ **伤害计算公式**
```
最终伤害 = 技能基础伤害 × (1 + 攻击力加成/100) × 属性克制 × 境界压制 × 暴击 - 防御力
```

### 2.4 生产系统

✅ **炼丹系统【MVP 核心】**
- 丹炉方块实体（Furnace Block Entity）
- 丹方系统（Recipe Manager）
- 品质判定（下品/中品/上品）
- 炼丹技能等级（LV1~LV10）
- 核心丹药：
  - **凝气丹**（练气期修炼加速）
  - **筑基丹**（练气→筑基，极稀缺）
  - **稳神丹**（-20% 走火风险）
  - **回灵丹**（恢复50%灵力）

✅ **炼器系统（基础）【MVP】**
- 炼器台方块
- 器形图纸系统
- 灵矿材料分级（玄铁矿等）
- 法器品阶：凡器 → 灵器 → 宝器
- **仅实现飞剑炼制路线**

✅ **制符系统（基础）【MVP】**
- 制符台方块
- 符纸 + 灵砂 + 朱砂
- 手动绘制符文（GUI 小游戏或简化为进度条）
- 核心符箓：
  - 火焰符（攻击）
  - 冰封符（控制）
  - 护体金光符（防御）
  - 瞬移遁符（移动）

### 2.5 世界内容

✅ **神秘小瓶系统【MVP 核心机制】**
- 唯一绑定物品，不可丢弃
- 每24小时积累1份灵液（离线计时）
- 最大5份（可通过任务升级到10/20）
- 使用效果：
  - 对植物使用：生长速度 ×10（蓝色灵液，初期）
  - **伪灵根/杂灵根必定获得，其他灵根极低概率**
- **天雷竹特殊交互**：×15 生长（为后期青竹蜂云剑铺路）

✅ **NPC 系统（核心 NPC）**
| NPC名 | 所在地 | 功能 |
|-------|-------|------|
| 墨老先生 | 大燕国村镇 | 灵根测试 |
| 七玄门七长老 | 七玄门 | 炼丹/炼器/功法/外务/阵法/战斗/秘密 |
| 李真人（黄枫谷长老）| 黄枫谷 | 玩家师傅（黄枫谷路线）|
| 厉飞羽 | 七玄门→散 | 同伴 NPC |

✅ **对话系统**
- JSON 对话树
- 条件分支（quest_stage, flags）
- 选项/行动触发

✅ **任务线**

**七玄门 arc（5阶段）：**
1. 测灵根：村镇集合 → 灵根测试 → 宗门收入
2. 入门：领黄龙功 → 劳役任务 → 学炼丹
3. 发现秘密：偷入密室 → 发现神秘小瓶 → 实验
4. 宗门内斗：长老勾结 → 收集证据 → 三岔路口选择
5. 离开大燕：天罡盟攻打 → 逃离 → 前往越国

**越国六大宗门 arc（5阶段，仅实现前3阶段）：**
1. 叩门考核：六门招募 → 选宗门 → 三关考核 → 外门弟子
2. 筑基困局：收集筑基丹材料 → **神秘小瓶种万年灵草** → 宗门任务积分
3. 内门竞争：年度选拔 → 竞争 → 叛徒调查支线

**后2阶段暂不实现（留给阶段二）：**
- ~~宗门外战~~
- ~~真传传功~~

✅ **灵地系统（简化版）**
- 普通灵脉节点：修为回速 ×1.3
- 宗门灵脉：×2.0（宗门专属）
- **地灵之眼、天然丹炉地暂不实现**

### 2.6 UI/HUD

✅ **基础 HUD**
- 左下角：境界名称、修为进度条、灵力条、神识范围、走火风险
- 右下角：技能栏（5个技能图标 + 冷却倒计时）
- 走火风险 >50% 时红色闪烁

✅ **打坐修炼 GUI**
- 实时修为增长显示
- 修炼速度加成列表
- 当前修为/修为上限

✅ **修仙属性界面（已有 CultivationStatsScreen）**
- 基础状态
- 战斗属性
- 灵根信息
- 功法信息
- 负面状态

## 四、MVP 实现优先级（按顺序）

根据 `ultivation-design-v3.md` §9.5 "阶段一 MVP" 排序：

1. ✅ 境界系统 + 修为/灵力属性 + 打坐修炼 GUI
2. ✅ 灵根测试 + 神秘小瓶基础功能（充能+植物加速）
3. ✅ 练气期全部技能 + 飞剑基础（单剑→三才剑阵）
4. ✅ 炼丹系统（一~三阶）+ 筑基丹稀缺机制
5. ✅ 七玄门 arc + 越国宗门 arc（加入到内门选拔为止）
6. ✅ 走火入魔机制
7. ✅ 基础 HUD（境界/修为/灵力/走火风险）

## 五、开发约束

### 5.1 技术约束

- **不修改现有 Java 代码**（本文档仅设计，不实现）
- **不扫描整个仓库**（仅基于设计文档提取）
- **只生成文档**
- Forge 1.20.1 + Java 17
- 使用 PlayerData Capability 存储
- 使用 Forge Event 系统

### 5.2 内容约束

- **不扩展设计文档内容**
- 严格按照【MVP】标签和"阶段一"提取
- 【扩展】内容全部延后到阶段二/三
- 原文中未明确标注 MVP 的高阶内容（金丹以上）延后

### 5.3 验收标准

玩家可以：
1. 测试灵根并获得对应修炼速度
2. 通过打坐从凡人修炼到筑基圆满
3. 学习并使用练气期9个技能 + 筑基期6个技能
4. 炼制凝气丹、筑基丹、稳神丹、回灵丹
5. 体验走火入魔机制并通过稳神丹/打坐降低风险
6. 完成七玄门完整任务线（5阶段）
7. 完成越国宗门任务线前3阶段（叩门→筑基困局→内门竞争）
8. 使用神秘小瓶加速万年灵草生长以获取筑基丹材料
9. 看到实时更新的 HUD（境界、修为、灵力、走火风险）
