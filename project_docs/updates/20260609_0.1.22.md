# 20260609 版本 0.1.22 更新记录

## 版本

- `mod_version` 从 `0.1.21` 提升到 `0.1.22`。

## 新增/调整

- 新增隐藏灵气浓度系统：按维度、地形/生物群系、隐藏灵脉与附近聚灵阵计算当前位置灵气浓度。
- 区域倍率第一版规则：普通区域约 1x，山脉/高地约 1.5x，深海岛屿约 2x，隐藏灵脉约 3x-5x，预留秘境/仙府维度约 10x。
- 下界标记为火煞魔气/炼体灵气，末地标记为虚空法则灵气，为后续炼体与法则感悟系统预留。
- 修炼 tick 接入灵气浓度：基础灵力收益按灵气倍率提升；蒲团修为经验使用平方根倍率提升。
- 新增 `spirit_detector` 测灵盘：右键显示当前区块灵气浓度、倍率来源和灵气性质。
- 新增 `leyline_compass` 寻脉罗盘：右键扫描附近隐藏灵脉核心，显示文字方向、距离与灵脉倍率，并记录目标坐标到物品 NBT。
- 新增 `spirit_gathering_array` 聚灵阵：放置后提高附近灵气浓度，最多统计 4 个阵盘。
- 补齐新物品/方块语言、模型、贴图、配方、掉落表与挖掘标签。
