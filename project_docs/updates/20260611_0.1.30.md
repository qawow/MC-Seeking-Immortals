# 2026-06-11 0.1.30 更新记录

## 本次目标

将用户上传的竖向参考图作为左侧技能栏 Overlay 的视觉外框/背景接入，同时不破坏 0.1.29 已实现的 B 方案独立全屏修仙面板。

## 修改内容

- 新增 GUI 纹理资源：`src/main/resources/assets/seeking_immortals/textures/gui/skill_bar_frame.png`。
- 将上传 JPG 自动裁剪到主体外框边界，尝试去除棋盘格背景并转为透明 PNG，再缩放到 GUI 友好的 86×256 资源尺寸。
- `TechniqueSkillBarOverlay` 新增 `ResourceLocation` 纹理绘制，渲染到 HUD 左侧。
- 7 个技能槽继续固定纵向排列，并按外框尺寸居中放入开口区域。
- 未同步时仍显示 7 个空槽；同步后继续显示占位图标、编号、技能首字与 tooltip。
- 外框绘制后恢复 `GuiGraphics#setColor(1,1,1,1)`，避免透明度/颜色污染后续 HUD。
- 版本号从 `0.1.29` 升到 `0.1.30`。

## 资源说明

源图是 JPG 且带棋盘格背景，本次使用自动阈值方式识别高亮近灰棋盘背景并生成 alpha。可用性已保留，但边缘可能存在轻微 JPG 压缩或抠图瑕疵；后续如有正式透明 PNG，可直接替换同路径资源。
