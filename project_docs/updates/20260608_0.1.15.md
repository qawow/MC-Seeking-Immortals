# 20260608 0.1.15 功法传承卷轴/书籍与按来源学习

## 目标

- 根据 `src/main/resources/data/seeking_immortals/cultivation` 下功法数据文件内的 `source` 字段创建对应功法传承卷轴/书籍。
- `special_common_techniques.json` 本次不读取、不生成来源传承。
- 使用对应传承卷轴/书籍后，让玩家学会该来源下满足条件的一系列技能。
- 将每条功法/术法的 `attribute` 字段作为使用条件。

## 已完成

- 新增 `TechniqueDataManager`：服务端从资源管理器读取 cultivation JSON，并显式排除 `special_common_techniques.json`。
- 新增 `TechniqueManualItem`：右键读取同来源技能，按 `attribute` 判断灵根属性条件，满足条件则写入玩家已学技能。
- `PlayerCultivation` 新增 `LearnedTechniques` 持久化集合，并保存/读取到 NBT。
- 从当前 cultivation 数据中生成并注册 60 个 `technique_manual_*` 传承物品，加入创造模式物品栏。
- 为 60 个传承物品补充中英文语言项和基础物品模型。
- 版本号更新为 `0.1.15`。

## 说明

- `attribute` 包含通用、辅助、神识、阵法、炼体、符、空间、封印等描述时默认可学。
- `attribute` 包含金/木/水/火/土/风/雷/冰/暗等属性时，需玩家灵根属性匹配。
- 当前完成的是“学会技能 ID”的基础系统，具体主动技能效果、施法 GUI、冷却、消耗与伤害结算仍留待后续。

- 结果：`BUILD SUCCESSFUL`。
