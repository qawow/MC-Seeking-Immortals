## 新增/调整

- 将蒲团打坐完善为“打坐吐纳”基础闭环：玩家手持蒲团放置，空手右键坐上蒲团后进入修炼状态。
- 坐在蒲团并保持打坐时，每 5 秒结算一次修为收益；灵力仍按每秒修炼 tick 恢复。
- 修为收益综合计算：当前位置灵气浓度平方根倍率 × 玩家修炼速度倍率 × 功法品质倍率 × 灵根/功法亲和倍率。
- 已学功法会从 `cultivation` JSON 数据中读取，并自动选择当前提供最高打坐收益的功法作为状态栏显示功法。
- 动作栏新增打坐状态显示：境界、修为进度、灵气吸收效率、当前功法、当前灵地。
- 中断条件补齐：受到攻击、移动或离开蒲团、饥饿过低、附近 8 格内有怪物时都会中断打坐吐纳。
- 补充中文/英文语言文本：受击中断、饥饿中断、怪物威胁中断、打坐状态栏。
