# 2026-06-08 更新：版本 0.1.2，境界扩展、灵根、寿元与突破公式

## 版本

- `mod_version` 从 `0.1.1` 提升到 `0.1.2`。
- 构建产物将变为 `seeking_immortals-0.1.2.jar`。

## 新增/实现

- 境界枚举扩展到炼气、筑基、结丹、元婴、化神。
- 炼气细分为 1-13 层，第 13 层为炼气大圆满。
- 筑基、结丹、元婴、化神细分为初/中/后期。
- 新增灵根枚举：伪灵根、四灵根、三灵根、双灵根、天灵根、异灵根。
- 新增灵根天赋系数，参与突破概率。
- 新增寿元字段 `lifespanYears`。
- 新增失败次数 `failedBreakthroughs`，用于保底值。
- 突破概率基础公式：基础概率 × 天赋系数 × 资源倍率 + 失败保底，上限 85%。
- `/seeking_immortals realm` 显示寿元和突破概率。
- 新增 `/seeking_immortals root` 查看灵根。

## 简化与待办

- 目前突破资源倍率仅接入破境丹辅助，具体丹药、功法、环境加成待后续实现。
- 失败惩罚目前简化为扣灵力、修为、寿元和增加失败次数。
