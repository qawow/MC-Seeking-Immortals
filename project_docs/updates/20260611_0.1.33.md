# 2026-06-11 更新：0.1.33 技能快捷键与单页修仙面板

## 修改内容

- 将 B 方案独立 `CultivationStatsScreen` 从基础、灵根、功法、负面状态四分页合并为一个单页总览。
- 移除/隐藏原四个分页按钮，页面内按“基础状态 / 灵根信息 / 功法信息 / 负面状态”分区展示。
- 保留从背包入口进入时返回背包、其他入口关闭页面的逻辑。
- 停止在打坐吐纳每 5 秒结算时调用 `showMeditationStatus` 发送动作栏状态提示，避免反复弹出大字/标题式提示；开始/停止/中断等必要提示保留。
- 新增 7 个技能释放按键，默认：槽 1~7 = `Z/X/C/G/B/N/M`。
- 新增 `ReleaseTechniquePacket`，客户端只发送槽位；服务端校验槽位范围、玩家已学技能、灵力是否足够后进行占位释放，并同步灵力。
- 新增 `TechniqueEditScreen` 和打开按键，默认 `R`；显示 7 个槽位、当前已学技能列表和“点击/拖拽绑定待实现”说明。
