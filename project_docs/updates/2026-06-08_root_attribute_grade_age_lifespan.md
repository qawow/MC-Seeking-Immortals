# 2026-06-08 更新：版本 0.1.3，灵根属性/品级、随机出生与寿元死亡

## 版本

- `mod_version` 从 `0.1.2` 提升到 `0.1.3`。
- 构建产物将变为 `seeking_immortals-0.1.3.jar`。

## 新增/实现

- 灵根拆分为属性与品级。
- 属性包括：金、木、水、火、土、风、雷、冰、无属性、仙灵根。
- 品级包括：下品、中品、上品、极品。
- 玩家首次登录随机生成灵根属性与品级。
- 灵根品级影响修炼速度和突破成功率。
- 灵根属性参与突破成功率。
- 新增年龄字段 `ageYears`。
- 世界每过一天，年龄增长 1 年。
- 年龄达到或超过寿元时，非创造/旁观玩家会因寿元耗尽死亡。

## 指令

- `/seeking_immortals root` 现在显示灵根属性、品级、突破系数、修炼速度系数。
- `/seeking_immortals realm` 现在显示年龄、寿元、剩余寿元、突破概率。
