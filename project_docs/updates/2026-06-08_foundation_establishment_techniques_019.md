# 2026-06-08 更新：版本 0.1.9，筑基期功法术法规则

## 版本

- `mod_version` 从 `0.1.8` 提升到 `0.1.9`。
- 构建产物目标为 `seeking_immortals-0.1.9.jar`。

## 新增规则数据

新增筑基期可用功法/术法规则数据：

- 数据文件：`src/main/resources/data/seeking_immortals/cultivation/foundation_establishment_techniques.json`
- 适用境界：筑基期 `FOUNDATION_ESTABLISHMENT`
- 字段包含：唯一 ID、名称、类型、属性倾向、来源出处、术法简介。

## 本次加入条目

- 巨剑术、千刃术、水罩术、土牢术、风缚之术、风墙术。
- 驱灵术、凝元功、归元功、弄焰诀、三转重元功（第一层）。
- 煞丹术、小五行须弥禁法、小须弥金刚阵、玄冰诀、御风决、血灵钻。

## 手册展示

新增 Patchouli 手册条目：`筑基期功法术法`，按攻防控制、修炼辅助、阵法遁术与禁术分组说明。

## 后续接入方向

- 功法学习系统：将筑基期规则作为炼气之后的进阶可学习条目。
- 灵根加成系统：根据金、水、土、风、火、冰、血、五行/空间等属性倾向接入加成。
- 宗门系统：青元剑诀、御灵宗、古剑门、魔道、阵法类可作为传承或掉落来源。
- 代价系统：煞丹术、血灵钻等魔道秘术后续应接入碎丹、重伤、心魔或寿元损耗等负面后果。
- 战斗系统：将攻击、防御、控制、御兽、阵法、遁术等类型映射到实际技能效果。
