# 2026-06-07 更新：灵石储量、逐秒修炼与 lingli 指令

## 新增

- 灵石加入 NBT 灵力储量。
- 灵石显示耐久条和 tooltip 描述。
- `/seeking_immortals lingli` 指令用于查看灵力、最大灵力和修为。
- 灵石吸收加入少量粒子、音效和动作栏进度提示。

## 调整

- 灵石必须拆成单个才能右键引导吸收，堆叠状态会提示不可用。
- 灵石右键引导吸收不再直接消耗整块灵石，而是消耗其 NBT 储量；储量耗尽后才消失。
- 普通状态下，主手/副手灵石逐秒消耗储量，提高灵力恢复。
- 打坐修炼状态下，主手/副手灵石逐秒消耗储量，提高修为增长。
- 修炼/恢复计算从每 5 秒改为每秒，数值按原方案缩小。
- 内部字段开始从 `qi` 迁移到 `spiritualPower`，保留兼容方法。
