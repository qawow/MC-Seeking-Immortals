## 版本

- `mod_version` 从 `0.1.20` 提升到 `0.1.21`。

## 新增/调整

- 阅读并参考 Patchouli 官方仓库/文档用法后，接入玩家首次进入世界自动获得 Patchouli《修仙引导手册》的逻辑。
- 使用 Patchouli 公开 API `PatchouliAPI.get().getBookStack(new ResourceLocation("seeking_immortals", "seeking_immortals_guide"))` 获取手册物品栈。
- 玩家首次获得手册后会写入玩家 `PersistentData` 标记 `SeekingImmortalsPatchouliGuideGiven`，避免每次登录重复发放；背包满时手册会掉落在玩家位置。
- 更新 Patchouli `book.json` 落地说明，改为说明首次入世自动发放手册。
- 新增语言键 `message.seeking_immortals.guide_book.given`。
- 将缺失正式贴图的 60 个 `technique_manual_*` 系列功法/术法传承物品全部详细列入 `project_docs/missing_and_placeholders.md`。
