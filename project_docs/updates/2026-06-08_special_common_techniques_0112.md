# 2026-06-08 更新：版本 0.1.12，特殊/通用术法规则

## 版本

- `mod_version` 从 `0.1.11` 提升到 `0.1.12`。
- 构建产物目标为 `seeking_immortals-0.1.12.jar`。

## 新增规则数据

新增特殊/通用术法规则数据：

- 数据文件：`src/main/resources/data/seeking_immortals/cultivation/special_common_techniques.json`
- 分类标识：特殊/通用术法 `SPECIAL_COMMON`
- 字段包含：唯一 ID、名称、类型、属性倾向、来源出处、术法简介。

## 本次加入条目

- 通用与法宝：灌注之法、封灵大法、通宝诀。
- 机关、御虫与灵纹：傀儡机关术、驱虫术、育虫心得、灵纹术。
- 鬼道魂术：凝魂术、炼魂术、牵魂术。
- 魔道与咒术：控神术、言咒之术。
- 阵法与轮回：阴火大阵、小五行须弥禁法、阴阳轮回决。

## 手册展示

新增 Patchouli 手册条目：`特殊/通用术法`，按通用与法宝、机关御虫与灵纹、魂术魔道与咒术、阵法与轮回分组说明。

## 后续接入方向

- 法宝系统：通宝诀可作为通天灵宝的专属驱动条件。
- 傀儡系统：傀儡机关术可接入傀儡制造、操控、灵识注入和战斗 AI。
- 御虫系统：驱虫术和育虫心得可接入灵虫捕捉、繁育、进阶和战斗指令。
- 魂魄系统：凝魂术、炼魂术、牵魂术、阴火大阵和阴阳轮回决可接入魂魄资源、魂魄反噬、召魂和轮回机制。
- 禁术代价系统：控神术、炼魂术等邪术应接入正道仇恨、业力、心魔、神魂损伤或阵营惩罚。
