# 2026-06-22 更新记录：0.1.48

## 本次重点

- 将 Codex 与 Claude Code 的固定执行顺序写入根目录 `AGENTS.md` / `CLAUDE.md`。
- 新增 `scripts/preflight.ps1`，用于在构建前检查代码/资源/构建逻辑变更是否同步 bump `mod_version`。
- 在 `build.gradle` 中新增 `aiPreflight` 任务，并让 `build` 依赖该任务。
- `build` 成功后记录 `.gradle/ai-preflight/last-success.json` 指纹，后续可发现“同版本号下继续改可发布文件”的情况。

## 版本与协议

- `gradle.properties`：`mod_version=0.1.48`
- `ModNetwork.PROTOCOL_VERSION`：维持 `4`
- 本次未修改 packet 字段、字段顺序或编码/解码格式。

## 固定流程摘要

1. 读 `project_docs/ai_handoff.md` 与 `project_docs/step_progress.md`。
2. 按 docs-only / shippable / network-incompatible 分类。
3. 修改已有文件前备份到 `.bak/<timestamp>/`。
4. 实现最小范围变更。
5. 需要时递增 `mod_version`；packet 不兼容时递增协议号。
6. 更新进度文档与版本日志。
7. 运行 `./gradlew build`。
8. build 失败则修复并重跑。
9. 最终报告包含备份、版本、协议与构建结果。
