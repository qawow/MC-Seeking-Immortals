# 20260608 0.1.16 传承卷轴占位图标与右侧技能栏

## 目标

- 将所有功法传承卷轴/书籍图标统一换成占位符，后续由用户补正式贴图。
- 将传承卷轴/书籍物品描述调整为固定四行：传承来源、使用条件、学会术法、右键使用。
- 在玩家屏幕右侧添加竖向技能栏，一个已学技能对应一个小方格。

## 已完成

- 60 个 `technique_manual_*` 物品模型统一指向 `seeking_immortals:item/technique_manual_placeholder`。
- 新增占位贴图 `textures/item/technique_manual_placeholder.png`。
- `TechniqueManualItem` 的 Tooltip 改为四行格式。
- 新增 `SyncLearnedTechniquesPacket`，登录和学习新术法后同步 `LearnedTechniques` 到客户端。
- 新增客户端 `TechniqueSkillBarOverlay`：在屏幕右侧按竖排绘制已学技能小方格，最多显示 10 个，超出显示 `+N`。
- 版本号更新为 `0.1.16`。

## 说明

- 当前技能栏为基础显示版：小方格使用技能 ID 哈希色与序号/首字母占位，不含正式技能图标。
- 技能栏暂未支持点击、快捷键释放、冷却遮罩、技能详情悬浮提示。
- 服务端仍以玩家 Capability/NBT 中的 `LearnedTechniques` 为准，客户端只接收显示用同步列表。

## 验证

- 已执行 `./gradlew build`。
- 结果：`BUILD SUCCESSFUL`。
