# 2026-06-08 更新：版本 0.1.5，灵根规则细化

## 版本

- `mod_version` 从 `0.1.4` 提升到 `0.1.5`。
- 构建产物目标为 `seeking_immortals-0.1.5.jar`。

## 新增/调整

- 天灵根改为五星极稀有资质，随机时只会生成金、木、水、火、土五行单属性。
- 天灵根修炼倍率提升到约 2.5，突破倍率提升到约 2.6，用于表现普通灵根 2-3 倍修炼速度与极低结丹瓶颈。
- 新增 `HIDDEN` 隐灵根类型，星级为六星。
- 新增隐灵根属性分支：隐雷、隐暗。
- 隐灵根暂不参与首次登录随机，只作为未来血脉、验灵阵、特殊机缘觉醒入口。
- 变异灵根统一命名，随机属性为雷、冰、风、暗。
- 雷、冰、风、暗分别补充战斗定位说明，并提高突破属性系数。
- 双灵根、三灵根定位为真灵根；四灵根、五灵根定位为伪灵根。
- 四灵根、五灵根的修炼倍率和突破倍率进一步压低，以贴合筑基率低、法力杂乱的设定。

## 暂未实现

- 补天丹暂未作为物品实装，本次仅作为后续待办：用于减少多余属性、提升灵根纯度、提高筑基/结丹成功率。
- 隐灵根觉醒机制暂未实装。
- 各属性对应功法、法术、炼丹、炼器、治疗、防御、暗杀/诅咒加成暂未实装。
