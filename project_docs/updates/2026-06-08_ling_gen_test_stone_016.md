# 2026-06-08 更新：版本 0.1.6，灵根测试石与检测逻辑

## 版本

- `mod_version` 从 `0.1.5` 提升到 `0.1.6`。
- 构建产物目标为 `seeking_immortals-0.1.6.jar`。

## 新增

- 新增灵根测试石物品：`ling_gen_test_stone`。
- 新增 `LingGenCalculator`，负责加权随机生成灵根类型、属性组合、纯度和觉醒状态。
- 灵根测试石支持右键自己或右键其他玩家。
- 检测优先消耗 1 块下品灵石；若没有灵石，则消耗检测石耐久。
- 检测结果以多行消息显示，模拟简易结果界面：
  - 灵根名称
  - 稀有度星级
  - 灵根属性
  - 纯度
  - 修炼速度倍率
  - 突破倍率
  - 功法/发展提示
- 检测时播放附魔粒子、末地烛粒子和紫水晶音效。
- 新增灵根纯度、已觉醒状态、已检测状态，并保存进现有玩家修炼 Capability/NBT。
- `/seeking_immortals root` 现在显示纯度、觉醒状态和星级。
- Curios 灵力护符被动灵力加成现在受灵根修炼倍率影响。
- 新增 32×32 像素风灵根测试石贴图。
- 新增物品模型、合成配方和中文语言文本。

## 加权随机规则

- 天灵根：约 0.1%-0.5%。
- 变异灵根：约 1%-3%。
- 真灵根（双/三）：约 20%-40%。
- 伪灵根（四/五）：约 60%-80%。
- 补天丹/任务修正因子已在计算器中预留接口，但补天丹物品仍未实装。

## 说明

- 本次未额外创建独立 `LingGenCapability`，而是复用现有 `CultivationProvider` 保存灵根数据，避免同一玩家出现两份灵根状态。
- 真正客户端 GUI 暂未实现；当前使用服务端安全的多行消息模拟结果弹窗。
