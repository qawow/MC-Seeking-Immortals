# 20260611 版本 0.1.31 更新记录

## 修改内容

- 新增修炼进度条 GUI 资源：`src/main/resources/assets/seeking_immortals/textures/gui/cultivation_progress_bar.png`。
- `BreathingHudOverlay` 的打坐吐纳 5 秒结算进度改用新修炼进度条纹理显示。
- `TechniqueSkillBarOverlay` 的左边距从 `4` 调整为 `1`，并把槽位布局宽度改为只围绕槽位本体计算，使 7 个技能槽更靠近屏幕左侧。
- 停止绘制 0.1.30 导入的 `skill_bar_frame.png` 外框，移除玩家左上角额外出现的本模组装饰图案。
- 版本号从 `0.1.30` 更新为 `0.1.31`。

## 左上角图案根因

本次复查所有客户端 HUD/Overlay 注册与 GUI 纹理绘制，当前本模组仅注册：

- `technique_skill_bar`：左侧技能栏。
- `breathing_hud`：打坐吐纳 HUD。

玩家左上角图案来源为 0.1.30 在 `TechniqueSkillBarOverlay` 中通过 `GuiGraphics#blit` 绘制的整张 `textures/gui/skill_bar_frame.png`。该外框位于屏幕左侧 `LEFT_MARGIN=4`、`TOP_OFFSET=44` 起始处，顶部装饰容易被理解为玩家左上角额外图案。本版不再绘制该外框，避免误显示；技能槽、tooltip 与吐纳进度条不受影响。

## 资源处理说明

用户源图为 JPG，带棋盘格背景。本次自动裁剪第三条完整进度条，并通过边缘泛洪方式将接近白/灰的棋盘格透明化。由于 JPG 压缩与抗锯齿会污染边缘，生成 PNG 可能仍有轻微浅色残留；若后续提供原始透明 PNG，可直接替换该资源。
